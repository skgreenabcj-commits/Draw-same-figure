# ずけい れんしゅう 🐱
```markdown
子ども向けの図形描画練習ウェブアプリです。AIが生成したお手本と同じ線を描くことで、
座標・図形感覚を楽しく身につけられます。

```
---

## 📖 概要

| 項目 | 内容 |
|------|------|
| 対象年齢 | 幼児〜小学校低学年 |
| プレイ形式 | お手本と同じ線分をグリッド上に描く（5問1セット） |
| AI問題生成 | Gemini API キーを設定すると毎回新しい問題を自動生成 |
| オフライン対応 | APIキー未設定時は組み込み問題バンク（各レベル10問）を使用 |
| 動作環境 | モダンブラウザ（Chrome / Safari / Firefox / Edge） / スマートフォン対応 |

---

| レベル２ | 4×4 | 4本 | なし | 2〜5 | 中 |

## 🤖 AI問題生成（Gemini API）

### APIキーの取得

1. [Google AI Studio](https://aistudio.google.com/) にアクセス
2. **Get API key** からAPIキーを発行
3. ゲーム画面またはadmin.htmlでAPIキーを設定

### 設定方法

**ゲーム画面から設定する場合**

1. スタート画面の「AIで問題を作る（Gemini APIキー）」をタップ
2. APIキーを入力して「保存」をクリック

**管理者画面から設定する場合（推奨）**

1. スタート画面の「AIモデル設定（管理者）」をクリック
2. `admin.html` が別タブで開きます
3. 「Gemini APIキー設定」セクションでAPIキーを保存
4. 「AIモデル優先順位」セクションで「利用可能モデルを取得」をクリック
5. 第1〜第3優先モデルをプルダウンで選択し「優先順位を保存」をクリック

> **APIキーはブラウザのlocalStorageにのみ保存され、外部サーバーへは送信されません。**

## 🛠️ 管理者設定画面（admin.html）

### 機能一覧

| セクション | 内容 |
|------------|------|
| Gemini APIキー設定 | キーの保存・クリア・表示切り替え |
| AIモデル優先順位 | 第1〜第3優先モデルのプルダウン選択・保存・リセット |
| 実行ログ・エラー状況 | 累計エラー回数・直近の成功モデル・エラー詳細・推奨アクション表示 |

### モデル選択のポイント

- **★推奨** と表示される `flash-lite` 系モデルを第1優先に設定してください
- `*-pro` / `*-image` 系モデルはフィルタリングにより表示されません
- 設定したモデルが廃止された場合、エラーバナーが表示され自動的にフォールバックします

## ⚙️ 技術仕様

### AIモデル管理（gemini.js v5.8）

```
優先チェーン（デフォルト）:
  1位: gemini-2.5-flash-lite
  2位: gemini-3.1-flash-lite-preview
  3位: gemini-2.5-flash
```

- モデル一覧は Gemini API から動的取得し、1時間キャッシュ
- `*-pro` / `*-image` 系はフィルタで除外
- 廃止モデル検出時は生きているモデルへ自動フォールバック
- 全モデル失敗時は組み込み問題バンクで代替
- API呼び出しタイムアウト: 8秒/回、最大30秒/セッション

### ステータス永続化

ゲーム画面（index.html）でのAI生成ログは `gemini_admin_status_v1` キーで
`localStorage` に書き込まれ、admin.html を開くと最新状態を参照できます。

### localStorageキー一覧

| キー | 用途 | TTL |
|------|------|-----|
| `gemini_api_key` | APIキー | なし（手動クリアのみ） |
| `gemini_admin_chain_v1` | 管理者設定モデルチェーン | なし |
| `gemini_model_v3` | 直近の成功モデルキャッシュ | 24時間 |
| `gemini_live_models_v1` | ライブモデル一覧キャッシュ | 1時間 |
| `gemini_admin_status_v1` | エラー・成功ステータス共有 | なし |
| `gemini_no_mime_v1` | JSONモード非対応モデルキャッシュ | 7日間 |
| `admin_log_v1` | 管理者操作ログ（最大50件） | なし |

## 🔧 トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| AIが問題を生成しない | APIキー未設定または無効 | admin.html でAPIキーを確認・再設定 |
| 赤いエラーバナーが表示される | モデルエラーまたは廃止検出 | バナーをタップして admin.html を開き、モデルを再設定 |
| 問題生成に時間がかかる | 第1優先モデルが低速 | admin.html で `flash-lite` 系モデルを第1優先に設定 |
| 「レート制限」エラーが続く | APIのリクエスト上限超過 | しばらく待ってから再試行 |
| モデル一覧が空になる | ネットワークエラーまたはAPIキー不正 | ネットワーク確認後、APIキーを再設定して「モデルを取得」 |
