# 🐱 ずけい れんしゅう — 図形模写学習アプリ

小学生受験対策向け「図形模写」練習 Web アプリです。  
ドット（グリッド）上の見本図形を見ながら、同じ図形をタッチ/マウスで描くことで「空間認識力・図形再現力」を楽しく鍛えます。

---

## ✅ 実装済み機能

| 機能 | 詳細 |
|------|------|
| 3段階レベル | Lv1(4×4・ヒント線2本) / Lv2(4×4・完全回答) / Lv3(5×5・複雑図形) |
| 見本表示 | 左パネルにドットグリッド＋正解線を表示 |
| タッチ/マウス操作 | 点へのスナップ付きドラッグで直線を引く |
| プレビュー線 | ドラッグ中に破線でプレビューを表示 |
| undo / clear | 1本ずつ取り消し・全消し |
| 正誤判定 | 線の向き双方向を考慮した完全一致判定 |
| 正解ポップアップ | gotit.png + 褒め言葉でアニメーション表示 |
| 不正解フィードバック | 赤い正解線をオーバーレイ表示 |
| 5問セッション | ランダム5問を順番に出題、スコア集計 |
| 結果画面 | 5問終了後にスコアとメッセージを表示 |
| Gemini AI問題生成 | APIキー入力でオリジナル問題を生成 |
| レスポンシブ | スマートフォン・タブレット対応 |
| Retina対応 | devicePixelRatio を考慮した高解像度描画 |

## 📁 ファイル構成

```
index.html          メイン HTML
css/
  style.css         スタイルシート（変数・アニメーション含む）
js/
  problems.js       内蔵問題バンク (各レベル8問) + ランダム選択
  canvas.js         Canvas 描画・タッチ/マウス操作
  gemini.js         Gemini API 連携（問題自動生成）
  app.js            アプリ全体の制御ロジック
images/
  cat-mascot.png    スタート画面キャラクター
  gotit.png         正解時お祝い画像
```

## 🎮 学習の流れ

```
スタート画面
  └── レベル選択 (Lv1/Lv2/Lv3)
       └── [任意] Gemini APIキー入力
            └── 「はじめる！」ボタン
                 └── 問題画面 (5問繰り返し)
                      ├── 見本を見てタッチ/ドラッグで線を引く
                      ├── 「こたえあわせ」ボタン
                      │    ├── 正解 → gotit.png + 褒め言葉 → 次の問題
                      │    └── 不正解 → 赤い正解線 → 次の問題
                      └── 5問終了 → 結果画面
```

## 📐 問題フォーマット（座標系）

```json
{
  "level": 1,
  "grid": { "cols": 4, "rows": 4 },
  "lines": [
    { "x1": 0, "y1": 0, "x2": 2, "y2": 0 },
    { "x1": 2, "y1": 0, "x2": 2, "y2": 2 }
  ],
  "hintLines": [
    { "x1": 0, "y1": 0, "x2": 2, "y2": 0 }
  ]
}
```

- 座標は **0始まりの整数インデックス**（点のグリッド位置）
- Level1/2: cols=rows=4 → x,y ∈ {0,1,2,3}
- Level3: cols=rows=5 → x,y ∈ {0,1,2,3,4}
- `hintLines` は Level1 のみ：事前に薄い青で描画される

## 🤖 Gemini AI 問題生成

1. スタート画面で「AIで問題を作る」をクリック
2. Gemini API キー（`AIza...`）を入力して「保存」
3. キーはブラウザの `localStorage` に保存（サーバー送信なし）
4. 「はじめる！」でゲーム開始すると自動的に5問を生成

**使用モデル**: `gemini-2.5-flash`  
**エンドポイント**: `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash:generateContent`

## 🔧 カスタマイズ方法

### 内蔵問題の追加
`js/problems.js` の `PROBLEM_BANK.level1 / level2 / level3` 配列に  
上記フォーマットで問題オブジェクトを追加するだけです。

### スタイル変更
`css/style.css` の `:root` CSS変数を変更することで  
カラーテーマ・フォント・余白などを一括変更できます。
